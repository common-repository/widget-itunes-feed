== Description ==
Show iTunes feed like apple music, iTunes music, ios apps ... on wordpress widget

== How to use ==

1. Open Apple Rss Generator [https://rss.itunes.apple.com/en-us](https://rss.itunes.apple.com/en-us)

2. Choose your setting, remember, only choose "Format" is JSON.

3. Copy Feed Url to "Feed Url" setting on widget config.

4. Save and enjoy.

=== Override support ===

You can create layout on 'layouts' folder, or create new layout at your theme at directory

[theme_directory] / widget-itunes-feed / layout.php

Then pick layout you want at widget setting
